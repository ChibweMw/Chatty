Chatty
=====================
A Realtime websocket powered chat application. It supports images, and custom usernames, too!

# Get Chatty
```

Install the dependencies and start the server.

```
npm install
npm start
go to http://localhost:3000
```

### Dependencies

* babel-core
* babel-loader
* babel-preset-es2015
* babel-preset-react
* css-loader
* node-sass
* sass-loader
* sockjs-client
* style-loader
* webpack
* webpack-dev-server
* react
* react-dom

* express
* ws
* uuid
